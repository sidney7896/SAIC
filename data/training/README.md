# Training Data

Store session-level or week-level bench training data here.

Recommended principles:

- keep the log format stable
- capture only fields that improve recommendation quality
- distinguish objective fields from subjective fields

Suggested core fields:

- date
- exercise
- set number
- load
- reps
- RPE or proximity to failure
- session readiness
- pain or irritation
- notes

See `bench-log.template.csv` for a starting format.
