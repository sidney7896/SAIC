# Research Vault

Store bench-specific source material here.

Recommended file types:

- deep research summaries
- methodology notes
- competition or technique notes
- recovery and fatigue research relevant to bench
- progression models worth testing

Recommended structure:

- one topic per file
- short title
- clear source notes if relevant
- summary first, raw notes after

Suggested naming examples:

- `01-bench-frequency-notes.md`
- `02-fatigue-and-volume.md`
- `03-competition-arch-setup.md`
- `04-deload-triggers.md`
