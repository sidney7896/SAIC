# ADR-001: Bench-First Scope

## Status

Accepted

## Decision

Phase 1 of the project is strictly focused on building the best possible private AI bench coach for one athlete.

## Rationale

- the main goal is bench performance, not breadth
- narrow scope increases coaching quality
- a single-user system can be much more personal and opinionated
- future expansion should rest on a strong bench-specific foundation

## Consequences

- multi-user concerns are deferred
- general wellness features are deferred
- broad workout programming is deferred unless directly needed for bench outcomes
- success is judged by recommendation quality and real-world usefulness, not feature count
